# mc-processor 
MC processor for trigger electronics evaluation using FPGA.

poisson-generator.vhd: Simulates poisson process.

pulse-delay.vhd: High resolution pulse delay.

bin-onehot.vhd: Binary to one-hot encoder.

density-generator.vhd: Inverse sampling from a given cummulative distribution.

control-generator.vhd: Control FSM for one channel.

Project is integratad in single-generator.vhd and multi-channel version event-generator.vhd
